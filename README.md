# 剧本大师 (Script Master) v2.0

从**一句话创意**到**工业级剧本 + AI 视频提示词**的 5 阶段全链路流水线。

融合 NeoWOW 核心方法论，专为 AI 视频创作（Seedance 2.0 / 即梦）设计。

## 5 阶段流水线

```
策划 → 编剧 (6步引擎) → 评估 (7维+自检) → 润色 (修复循环) → 导演 (分镜)
```

## 核心特性

- **6步精修优化引擎**：结构化分析 → 视觉锚定 → 全局策略 → 镜头分配 → 构图设计 → 优化结果
- **12维镜头设计体系**：景别/FG-MG-BG/光效/明度/QC/角度/构图/画面5要素/运镜/理论/节拍/AI提示词
- **FG/MG/BG 三层构图**：前景/中景/背景独立设计
- **电影理论引用库**：经典电影技法支撑
- **工业级 QC 检查**：7 维度质量评估 + 自检
- **修复反馈循环**：评估→润色→修复自动循环
- **快慢双模**：快速出稿/深度精修两种模式
