# 剧本大师 v2.0 — AI视频剧本工坊

## 定位

从**一句话创意**到**工业级剧本+AI视频提示词**的 5 阶段全链路流水线。

v2.0 融合了 NeoWOW 的核心方法论，新增 **6步精修优化引擎**、**12维镜头设计体系**、**FG/MG/BG 三层构图**、**电影理论引用库**、**工业级QC检查**、**修复反馈循环**。

---

## 核心工作流（5阶段）

```
用户输入 ──→ [策划师] ──→ [编剧(6步引擎)] ──→ [评估师(7维+自检)]
                                                     │
                                                     ▼
                        [导演(分镜提示词)] ←── [润色师(标准+修复)]
```

### 阶段速览

| 阶段 | 角色 | 核心任务 | 产出 |
|------|------|---------|------|
| 1 | 策划师 | 创意→人物/世界观/情节点 | 00-项目元数据 + 01-人物设定 + 02-场景大纲 + 03-情节点大纲 |
| 2 | **编剧** | **6步精修优化引擎** | 04-剧本草稿（含12维镜头设计） |
| 3 | 评估师 | 7维评分 + 时间轴/空间自检 + 视觉平衡扫描 | 05-评估报告 |
| 4 | 润色师 | 对白精修 + 修复模式(条件触发) | 06-终稿剧本 |
| 5 | **导演（新）** | 终稿→Seedance/Kling分镜提示词 | 07-分镜提示词表 |

---

## v2.0 核心升级亮点

### 🔥 6步精修优化引擎（编剧角色）
```
剧本结构化分析 → 视觉锚定提取 → 全局视觉策略 
→ 镜头分配计划 → 镜头构图设计 → 优化结果(AI提示词)
```
支持快速模式(4步)和精修模式(6步)。

### 🎬 12维镜头设计体系
每个镜头标准化产出：景别/FG-MG-BG/光效/明度/QC/角度/构图/画面5要素/运镜/理论依据/故事节拍/AI提示词

### 📐 FG/MG/BG 三层构图 + QC检查
- 前景(Foreground) / 中景(Midground) / 背景(Background) 标准化描述
- 剪影检查 + 切线检查（工业级质量控制）
- 明度构成 Value 值标注

### 🎥 电影理论引用库
每镜标注理论依据 + 参考经典电影（《阿拉伯的劳伦斯》《老无所依》等）

### 🔍 两步自检体系（评估师）
- 时间轴与空间逻辑闭环
- 视觉平衡扫描（景别/节奏/色彩/运镜）

### 🔧 修复反馈循环（润色师）
自检发现问题 → 自动激活修复模式 → 修复后回写

## 交互模式

| 模式 | 说明 | 触发方式 |
|------|------|---------|
| **自动模式** | 5阶段自动依次执行 | 默认 |
| **交互模式** | 每阶段完成后暂停，等待确认 | "逐阶段来" / "一步步来" |
| **恢复模式** | 从指定阶段继续 | "继续上次" / "从第X阶段继续" |

---

## 优化模式（编剧阶段）

| 模式 | 步骤数 | 说明 |
|------|--------|------|
| 精修模式（默认） | 6步 | 完整优化，保障高质量输出 |
| 快速模式 | 4步 | 精简流程，快速产出初稿 |

## 状态管理

1. 每个阶段完成后立即写 MemPalace
2. 项目ID：`{YYYYMMDDHHMM}-{一句话主题}`
3. 通过 `06-终稿剧本.md` 判断完成状态
4. 中途修改后，后续阶段自动重新执行

## 错误处理

- 用户输入模糊 → 策划师提3个澄清问题
- 上下文丢失 → 从 MemPalace 读取上一阶段产出
- 评估分数 < 7.5 → 自动建议回到策划阶段
- 用户中断 → 当前阶段已写入 MemPalace，说"继续"即可恢复
