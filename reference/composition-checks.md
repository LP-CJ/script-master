# FG/MG/BG 构图 + QC 检查规范

> 12维镜头设计中的构图标准化和工业级质量控制体系。

---

## 一、FG/MG/BG 三层构图标准模板

每个镜头的构图描述必须覆盖三层：

```
┌─────────────────────────────────────┐
│ FG (Foreground / 前景)              │
│ · 最靠近镜头的元素                    │
│ · 通常虚化处理（浅景深）              │
│ · 功能：增加画面深度/框架/窥视感        │
├─────────────────────────────────────┤
│ MG (Midground / 中景)               │
│ · 画面主体所在层                     │
│ · 人物/核心动作/关键道具              │
│ · 功能：承载叙事信息                  │
├─────────────────────────────────────┤
│ BG (Background / 背景)              │
│ · 最远离镜头的元素                    │
│ · 环境/氛围/空间信息                  │
│ · 功能：建立场景/强化情绪              │
└─────────────────────────────────────┘
```

### 标准模板

```
▸ FG: [前景元素描述，材质/虚化状态]
▸ MG: [主体位置 + 动作状态，占据画面比例]
▸ BG: [环境描述 + 色调 + 景深状态]
```

### 示例（西部片沙暴场景）

```
▸ FG: 画面左下角有被风沙掩埋一半的枯萎灌木残骸，作为深度参考点。浅景深虚化。
▸ MG: 黑马载着男主从画面右侧向左中心缓慢移动，人马合一构成深色三角形剪影。占据画面40%。
▸ BG: 漫天飞舞的黄沙形成半透明幕帘效果，通过景深控制使远景维持模糊的土黄色。
```

## 二、剪影检查 (Silhouette Check)

### 检查规则

| 检查项 | 合格标准 | 不合格表现 |
|--------|---------|-----------|
| 主体轮廓 | 主体深色块与背景亮色区形成清晰分离 | 主体边缘融入背景，无法辨识轮廓 |
| 头肩分离 | 角色头部轮廓与背景有明显明度差 | 深色头发融入深色背景 |
| 动作可读 | 关键动作的剪影瞬间可辨识 | 手脚与身体剪影重叠，动作不可读 |
| 道具分离 | 手持道具与身体剪影有分离空间 | 武器/道具被身体剪影吞没 |

### 输出格式
```
▸ 剪影检查: [PASS/FAIL]
  - 合格: [通过与原因]
  - 风险: [具体问题描述]（如FAIL）
```

## 三、切线检查 (Tangent Check)

### 检查规则

| 切线类型 | 说明 | 判定 |
|---------|------|------|
| **边缘切线 (Edge Tangent)** | 人物轮廓与画框边缘重合 | FAIL — 留15%呼吸空间 |
| **重叠切线 (Merge Tangent)** | 两人物边缘重叠 | FAIL — 保持分离 |
| **隐藏切线 (Hidden Tangent)** | 人物边缘与背景线条重合 | FAIL — 错开或虚化 |
| **关节切割 (Joint Cut)** | 切割线位于膝/肘/颈等关节 | FAIL — 偏移5%以上 |

### 输出格式
```
▸ 切线检查: [PASS/FAIL]
  - 切割线位置: [描述]
  - 关节回避: [是/否]
  - 画框间距: [描述]
```

## 四、明度构成 (Value Structure)

### Value 值标尺

```
V1 ─ V2 ─ V3 ─ V4 ─ V5 ─ V6 ─ V7 ─ V8 ─ V9 ─ V10
黑                  中灰                    白
```

### 输出格式
```
▸ 明度构成:
  - 风格: [暗调(Low Key)/中间调(Mid Key)/亮调(High Key)]
  - 分布: [主体V值]/[环境主调V值]/[高光区V值]
  - 反差: [低/中/高]
```

## 五、景别分布检查清单（评估师使用）

全剧完成后，检查景别分布是否合理：

| 检查项 | 健康范围 |
|--------|---------|
| ELS/VWS/WS 占比 | 10-20%（开场+转场） |
| MS 占比 | 30-40%（叙事主力） |
| MCU 占比 | 15-25%（情绪过渡） |
| CU/ECU 占比 | 10-20%（情绪高潮） |
| 连续3个以上同景别 | 应避免（产生视觉疲劳） |

## 六、色彩/影调统一性检查

| 检查项 | 说明 |
|--------|------|
| 全剧主色调 | 不超过3种主色，贯穿全剧 |
| 幕间过渡色 | 每幕可有色调偏移，但需有过渡 |
| 情绪对色 | 压抑场景=冷/暗，爆发场景=暖/亮 |
| 影调连续性 | 相邻镜头Value值跨幅不超过3级 |
