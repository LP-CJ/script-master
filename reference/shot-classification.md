# 景别/角度/运镜分类词典

> 12维镜头设计中使用的标准化术语，中英对照。

---

## 景别 (Shot Size)

| 中文 | 英文 | 缩写 | 取景范围 | 叙事用途 |
|------|------|------|---------|---------|
| 极端远景 | Extreme Long Shot | ELS | 极远距离，人物极小 | 建立史诗感/环境压迫 |
| 大远景 | Very Wide Shot | VWS | 远距离，人物清晰但小 | 交代场景全貌 |
| 远景 | Wide Shot / Long Shot | WS/LS | 人物全身+环境 | 建立人物与环境关系 |
| 中全景 | Medium Long Shot | MLS | 膝盖以上 | 展示肢体语言 |
| 中景 | Medium Shot | MS | 腰部以上 | 标准叙事景别 |
| 中近景 | Medium Close-Up | MCU | 胸部以上 | 开始聚焦情绪 |
| 近景 | Close-Up | CU | 肩部以上/面部 | 揭示情绪细节 |
| 极特写 | Extreme Close-Up | ECU | 眼睛/嘴唇/手指 | 情绪爆发峰值 |
| 插入特写 | Insert Shot | INSERT | 道具/手部/文字 | 叙事关键信息 |

## 镜头角度 (Camera Angle)

### 水平角度 (Horizontal Angle)
| 中文 | 英文 | 说明 |
|------|------|------|
| 正面 | Front | 角色正对镜头，建立直接联系 |
| 3/4正面 | 3/4 Front | 最常见的叙事角度，看到面部+部分侧脸 |
| 1/3侧面 | 1/3 Side | 接近侧面，增加神秘感 |
| 侧面 | Side / Profile | 完全侧面，观察者视角 |
| 3/4背面 | 3/4 Back | 跟随感，即将揭示 |
| 背面 | Back | 未知/悬念/孤独 |

### 垂直角度 (Vertical Angle)
| 中文 | 英文 | 说明 |
|------|------|------|
| 平视 | Eye Level | 与角色视线平齐，中立客观 |
| 轻微仰拍 | Mild Low Angle | 略微仰视，增加存在感 |
| 中度仰拍 | Moderate Low Angle | 明显仰视，英雄化/威压 |
| 极端仰拍 | Extreme Low Angle | 极低角度，极度权威或压迫 |
| 轻微俯拍 | Mild High Angle | 略微俯视，增加脆弱感 |
| 中度俯拍 | Moderate High Angle | 明显俯视，上帝视角 |
| 极端俯拍 | Extreme High Angle / Bird's Eye | 鸟瞰，绝对控制/命运视角 |

## 运镜类型 (Camera Movement)

| 中文 | 英文 | 说明 | 情绪效果 |
|------|------|------|---------|
| 固定 | Static | 机位完全不动 | 稳定/压抑/沉思 |
| 推进 | Push In / Dolly In | 镜头向前推进 | 逼近/聚焦/紧张 |
| 拉远 | Pull Out / Dolly Out | 镜头向后拉远 | 离开/放手/升华 |
| 微推 | Micro Push | 极缓慢推进 | 凝视/情感积累 |
| 跟拍 | Follow / Tracking | 跟随主体移动 | 陪伴/代入/旅程 |
| 左摇 | Pan Left | 水平向左转 | 扫描/探索 |
| 右摇 | Pan Right | 水平向右转 | 扫描/探索 |
| 上摇 | Tilt Up | 垂直向上转 | 揭示/仰望 |
| 下摇 | Tilt Down | 垂直向下转 | 降落/俯视 |
| 升格 | Crane Up | 机位上升 | 离开/升华/宏大 |
| 降格 | Crane Down | 机位下降 | 进入/降临 |
| 手持 | Handheld | 手持晃动 | 真实/紧迫/混乱 |
| 稳定器 | Steadicam / Gimbal | 平滑移动 | 流畅/沉浸 |
| 变焦推 | Zoom In | 光学变焦推进 | 突然聚焦/冲击 |
| 变焦拉 | Zoom Out | 光学变焦拉远 | 揭示环境/反转 |
| 希区柯克变焦 | Dolly Zoom / Vertigo | 推拉+反向变焦 | 眩晕/心理失衡 |

## 运镜速度 (Movement Speed)

| 中文 | 英文 | 适用场景 |
|------|------|---------|
| 极慢 | Ultra Slow | 仪式感/凝视/沉思 |
| 缓慢 | Slow / Slow Motion | 压抑/紧张积累/美感 |
| 中速 | Medium | 正常叙事节奏 |
| 快速 | Fast | 追逐/动作/紧张 |
| 极快 | Ultra Fast / Whip | 冲击/混乱/突然 |

## 明度构成 (Value Structure)

| 级别 | 说明 | 视觉特征 |
|------|------|---------|
| V1 | 完全黑 | 无可见细节 |
| V2 | 极暗 | 仅轮廓可辨 |
| V3 | 暗部 | 大量阴影，低沉压抑 |
| V4 | 深灰 | 可辨认细节，暗调 |
| V5 | 中灰 | 中间调，正常曝光 |
| V6 | 中亮 | 稍微偏亮，通透感 |
| V7 | 亮部 | 明亮，高调 |
| V8 | 高亮 | 接近过曝，眩光效果 |
| V9 | 极亮 | 局部过曝，光晕 |
| V10 | 纯白 | 完全过曝 |

**明度风格**：
- **暗调 (Low Key)**：V1-V4主导，压抑/神秘/恐怖
- **中间调 (Mid Key)**：V4-V6主导，写实/叙事
- **亮调 (High Key)**：V7-V10主导，明亮/轻快/梦幻
