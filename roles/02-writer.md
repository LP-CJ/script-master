## 6步精修优化引擎（精修模式）

### 优化流程概览

```
┌─────────────────────────────────────────────────────┐
│ Step 1: 剧本结构化分析 (10%)                          │
│ Step 2: 视觉锚定提取 (15%)                            │
│ Step 3: 全局视觉策略 (20%)                            │
│ Step 4: 镜头分配计划 (35%)                            │
│ Step 5: 镜头构图设计 (65%)                            │

```

每步完成后显示进度百分比，让用户感知进度。

### Step 1: 剧本结构化分析

**目标**：解析剧本的叙事结构、场景划分、情节冲突点。

**操作**：
1. 读取情节点大纲，提取每个情节点的时间/地点/人物/事件
2. 识别关键转折点（hook / turning_point / climax）
3. 标注每场的情绪走向（与情节点大纲对齐）
4. 检查人物在每场的出场频率和戏剧功能

**输出**（追加到 `04-剧本草稿.md`）：
```markdown
## 结构化分析

| 场次 | 场景 | 人物 | 情节点类型 | 情绪走向 | 核心冲突 |
|------|------|------|-----------|---------|---------|
| 第1场 | XX | XX | hook | 期待↑ | XX |
...
```

**进度**：告知用户"✅ Step 1/6 完成 · 剧本结构化分析"。

---

### Step 2: 视觉锚定提取

**目标**：从剧本中提取核心视觉元素——关键物体、环境特征、色彩基调、光线母题。

**操作**：
1. 扫描剧本全文，提取反复出现的视觉元素（如：黄沙、残阳、左轮手枪）
2. 为每个视觉元素标记"视觉母题"（贯穿全剧的视觉符号）
3. 识别色彩/色调线索（压抑场景→冷暗，爆发场景→暖亮）
4. 识别光线母题（如：丁达尔光柱 = 希望/残酷）

**输出**（追加到 `04-剧本草稿.md`）：
```markdown
## 视觉锚点

| 视觉元素 | 类型 | 首次出现 | 出现频率 | 母题功能 |
|---------|------|---------|---------|---------|
| 黄沙 | 环境母题 | 第1场 | 全剧 | 压迫与宿命 |
| 左轮手枪 | 道具母题 | 第4场 | 3次 | 暴力与抉择 |
...
```

**进度**：告知用户"✅ Step 1-2/6 完成 · 视觉锚定提取"。

---

### Step 3: 全局视觉策略

**目标**：制定整章（或全剧）的色调、影调、镜头风格统一策略，确保视觉一致性。

**操作**：
1. 制定全剧主色调（不超过3种主色）
2. 按幕制定影调变化（第1幕→暗调压抑 / 第3幕→亮调爆发）
3. 确定镜头风格（写实/唯美/燃向/粗粝）
4. 确定运镜风格（手持晃动=真实 / 稳定器=史诗 / 固定=沉思）

**输出**（追加到 `04-剧本草稿.md`）：
```markdown
## 全局视觉策略

- **主色调**：土黄+灰褐+血红（西部片经典三色）
- **影调曲线**：第1幕(暗调V2-V5) → 第2幕(中调V4-V6) → 第3幕(高反差V1-V9)
- **镜头风格**：极致的粗粝感（Grit），大量颗粒感，无美化滤镜
- **运镜风格**：固定+缓慢推进为主，无快速剪辑
- **参考电影**：《不可饶恕》《老无所依》《荒野大镖客》
```

**进度**：告知用户"✅ Step 1-3/6 完成 · 全局视觉策略"。

---

### Step 4: 镜头分配计划

**目标**：根据剧本节奏，为每场分配镜头数量、时长、景别比例、情绪节奏。

**操作**：
1. 按情节点拆分镜头（一个情节点 = 1-3个镜头）
2. 分配每镜时长（按情绪节奏：开场长/高潮短/特写中）
3. 规划景别曲线（避免连续同景别，确保节奏起伏）
4. 预留情绪爆发镜头的特写配额

**输出**（追加到 `04-剧本草稿.md`）：
```markdown
## 镜头分配计划

| 镜号 | 情节对应 | 预估时长 | 建议景别 | 情绪峰值 | 人物 |
|------|---------|---------|---------|---------|------|
| #1 | hook-环境建立 | 5s | WS | 环境压迫 | 黑马/男主 |
| #2 | 丁达尔效应 | 0s | CU | 视觉冲击 | 无 |
| #3 | 跟随入镇 | 4s | MS | 代入感 | 男主 |
...

**景别分布**：WS:2(22%) | MS:2(22%) | CU:4(44%) | ECU:1(11%)
**总镜头数**：9 | **总时长**：45s
```

**进度**：告知用户"✅ Step 1-4/6 完成 · 镜头分配计划"。

### Step 5: 镜头构图设计

**目标**：为每个镜头设计 FG/MG/BG 三层构图、角度、光影、明度构成 + QC检查。

这是最核心的一步，每个镜头必须产出以下结构化字段：

#### 每镜必填项

```
┌──────────────────────────────────────────┐
│ 片场/场景/人物/道具                       │
│ 原始剧本 → 理论依据 → 故事节拍            │
│ 景别 + FG/MG/BG                          │
│ 光效 + 明度构成                           │
│ 剪影检查 + 切线检查 (QC)                  │
│ 角度 + 景别分类 + 角度依据                │
│ 构图描述 + 画面5要素                       │
│ 运镜类型+速度+依据+逐帧描述              │
└──────────────────────────────────────────┘
```

#### 设计参考

从 `reference/` 文件中调取：
- **film-theory-library.md**：为每镜选择理论依据和参考电影
- **shot-classification.md**：统一使用标准术语
- **composition-checks.md**：执行QC检查和明度标注

#### 输出格式

每个镜头按 `templates/04-镜头设计-template.json` 的结构展开，以 Markdown 形式写入：

```markdown
### 镜头 #1 · 5s

**片场**：入镇之路 | **场景**：小镇街道（外）
**人物**：男主 | **道具**：黑马

**原始脚本**：沙暴渐渐平息，阳光如同利剑般刺破沙尘...
**理论依据**：中度俯拍大远景建立上帝视角，展示小镇在自然伟力下的破败...
（参考：《阿拉伯的劳伦斯》沙漠远景）
**故事节拍**：沙暴平息后的小镇全貌

**景别**：远景(WS)
▸ FG：左下角质感粗糙、纹理干裂的深褐色木桩
▸ MG：男主骑在黑马上，形成深色剪影，从右侧向中心移动
▸ BG：低饱和度土黄色建筑群，阳光穿透沙尘形成丁达尔光柱

**光效**：高反差直射硬光 | **明度构成**：暗调(Low Key)，主体V3|环境V5|高光V8

**剪影检查**：PASS — 深色防尘大衣与土黄街道形强对比 | **切线检查**：PASS — 木桩与地平线有明确空间差

**角度**：正面+中度俯拍 | **景别分类**：远景(WS) — 建立宏观空间环境
**角度依据**：俯拍建立上帝视角，呈现小镇在自然下的渺小

**构图**：中度俯拍大远景，镜头展现一条笔直延伸的荒凉街道...

**画面**：人物(男主位于中景左侧微小剪影) | 姿态(跨坐马上，身形垂直稳定) | 环境(线性街道，黄沙铺地) | 道具(边缘可见风化木桩)

**运镜**：推进+下俯 | **速度**：缓慢
**运镜依据**：慢速推进配合下俯，视觉化"沙暴平息"的平稳感
**视频运镜**：镜头从高处缓慢向前推进并伴随平滑下俯动作，视觉中心从街道全貌逐渐向地面细节收缩。中速稳定节奏，5秒。

**Seedance提示词**：西部片风格，大远景俯瞰荒凉小镇，黄沙铺满街道，木板房排列两行，丁达尔光柱穿透沙尘斜射地面，深色剪影骑黑马缓慢进入，电影感，粗粝质感，16:9
```

**进度**：告知用户"✅ Step 1-5/6 完成 · 镜头构图设计 (X个镜头)"。

### Step 6: 优化结果

**目标**：组装所有分析结果，生成完整镜头设计文档 + 每个镜头的 AI 视频提示词。

**操作**：
1. 将所有镜头按 Step 5 的格式完整组装
2. 为每个镜头生成 Seedance 2.0 优化提示词
3. 检查所有 QC 项是否通过（FAIL 的标记为待修复）
4. 生成最终摘要（镜头总数/总时长/景别分布/QC通过率）

**输出**：完整的 `04-剧本草稿.md`，包含：
- 结构化分析表
- 视觉锚点表
- 全局视觉策略
- 镜头分配计划表
- 所有镜头的完整设计（每个含12维字段 + Seedance提示词）
- 优化摘要

## 快速模式（4步）

快速模式跳过 Step 3（全局视觉策略）和 Step 5（详细构图），改为：
1. 剧本结构化分析
2. 视觉锚定提取
3. 镜头分配计划（直接进）
4. 优化结果（简化版镜头描述 + 提示词）

快速模式下的镜头格式：
```markdown
### 镜头 #1 · 10s
**场景**：沙暴荒野（外） | **景别**：ELS | **运镜**：固定
**描述**：漫天黄沙填满全画幅，地平线被遮蔽，沙脊剧烈流动...
**Seedance提示词**：...
```

## 人物一致性约束

在全部步骤中，必须遵循：
- 每句对白符合人物 `speech_style`
- 主角不能违背 `core_desire` / `core_fear`
- 每个场景必须在 `场景大纲.json` 定义范围内
- 特殊道具使用需符合设定
