## 标准润色流程

### 第一步：按优先级处理评估问题

按评估师的优先级排序，逐场修改：

```
对于每个⚠️问题：
1. 定位到对应镜头
2. 按💡修改建议执行
3. 确保修改不引入新问题
4. 在终稿中标记 "[已修复]"
```

### 第二步：对白精修

**核心原则**：每句对白必须符合人物 `speech_style` 定位 + `core_desire`/`core_fear` 约束。

| 语风 | 特征 | 精修原则 |
|------|------|---------|
| CONFIDENT | 简洁有力，不解释 | 删除解释性后缀 |
| AGGRESSIVE | 短促，带威胁感 | 每句不超过10字 |
| GENTLE | 轻柔缓，有留白 | 留0.5-1秒的未说完感 |
| MYSTERIOUS | 意味深长 | 不直接回答问题 |
| COLD | 极度克制 | 零情绪形容词 |
| HUMOROUS | 轻松带梗 | 检查梗是否有效 |
| ENTHUSIASTIC | 热情 | 适当使用感叹 |
| SARCASTIC | 讽刺 | 表面夸奖实则贬损 |
| SHY | 害羞 | 结巴/停顿/后半句变小声 |
| FORMAL | 正式 | 使用敬语/书面语 |
| CHILDLIKE | 天真 | 短句/简单词/好奇心 |

**对白检查清单**：
- [ ] 主角每句对白符合 `speech_style`？
- [ ] 删除所有"自解释"对白（角色说完后自己解释刚才的话）
- [ ] 对白超过30字/句的拆分或精简
- [ ] 关键对白有"留白"（给观众想象空间）
- [ ] 配角对白服务于与主角的关系定位

### 第三步：情绪节奏终检

- 第1幕结束镜头 → 情绪："期待/好奇"，不能是"满足"
- 第2幕最低点 → 情绪："压抑/绝望"，爆发前的必要积压
- 第3幕爆发点 → 情绪："🔥爆发🔥"，视觉描述需匹配能量级别
- 任一情绪持续超过3个镜头 → 需要调节

### 第四步：QC最后审查

- 检查所有镜头的 `silhouette_check` 和 `tangent_check`
- FAIL 的镜头标注 `[QC待修复]`
- PASS 的镜头标注 `[QC通过]`

## 修复模式（条件触发）

**触发条件**：评估报告中发现 ⚠️ 问题。

**修复流程**：

```
1. 列出所有 ⚠️ 问题
2. 按优先级逐项修复
3. 每个修复项标注修复状态
4. 修复完成后自检：修复是否引入新问题？
5. 更新评估报告中的问题状态
```

### 修复状态标签

| 标签 | 含义 |
|------|------|
| `[已修复]` | 已按建议修改完成 |
| `[保留观察]` | 问题存在但暂不修改（给出理由） |
| `[需要讨论]` | 需要用户决策 |

### 修复后的自检

```markdown
## 🔧 修复报告

| 问题编号 | 位置 | 修复方式 | 修复状态 | 验证结果 |
|---------|------|---------|---------|---------|
| ⚠️1 | 镜头#3 | 调整景别 MCU→CU | [已修复] | ✅ 情绪更聚焦 |
| ⚠️2 | 镜头#6 | 增加过肩遮挡 | [已修复] | ✅ 增加神秘感 |
| ⚠️3 | 镜头#8 | 暂缓 | [保留观察] | 需用户看效果 |
```
